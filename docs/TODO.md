# 战略路线 - AI 媒体中心

不要和安卓 TvBox 拼资源站数量，低壁垒方向只作为基础能力维护。真正差异化路线是：

播放器 + AI + 本地媒体库 + 自动化 + Python 插件生态。

## S 级差异化能力

- [ ] AI 智能搜索：支持“类似黑镜的高分科幻”“节奏快的悬疑日剧”“适合晚上看的轻松电影”等自然语言检索
- [ ] 本地 Embedding / 向量搜索：融合本地媒体库、TMDB 元数据、插件搜索结果进行排序
- [ ] AI 元数据修正：用 LLM 修复混乱标题、文件名、季集、年份、版本信息
- [ ] AI 双语字幕：实时翻译并保留语气、角色口吻、动漫 / 影视翻译风格
- [ ] Whisper 自动字幕：本地语音识别、字幕生成、多语言翻译
- [ ] Python 插件生态：把完整 Python 能力作为核心护城河，而不是只做 JS / XPath / JSON 规则插件

## A 级桌面端优势

- [ ] NAS / 本地媒体中心：SMB / NFS / WebDAV / S3 / rclone 深度集成
- [ ] 自动媒体整理：下载后自动重命名、分类、海报墙、归档
- [ ] Web Remote / 下载系统 / 后台扫描服务
- [ ] 本地索引数据库：SQLite FTS5、向量索引、全文索引，支持搜台词 / 演员 / 场景

## B 级播放器增强

- [ ] HDR / Tone Mapping / 超分 / 插帧 / 色彩管理
- [ ] 音频增强
- [ ] 高级字幕系统：双语学习、生词点击、自动暂停、OCR 字幕提取
