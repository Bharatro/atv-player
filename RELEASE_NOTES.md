## 新增

- 新增内置 YouTube 标签页：检测到系统 `yt-dlp` 可用时显示，支持分类浏览、搜索、频道、播放列表和视频直接播放。
- 新增 YouTube 登录内容入口，支持通过 `yt-dlp` 读取账号订阅视频、订阅频道、播放历史和稍后再看。
- 新增 YouTube 分类配置源，支持内置分类、远程 URL 和本地 JSON / JSONC；配置兼容 TVBox `class` / `filters` 结构，并支持频道、播放列表、视频 id 和 `LIST:` 关键词分类。
- 新增 YouTube 默认字幕、默认音轨、元数据语言和地区设置，便于按偏好选择字幕、音轨和内容语言。

## 优化

- 优化 YouTube 打开速度，点击卡片后先用已有标题和封面打开播放器，再在后台解析真实播放地址。
- 优化 YouTube 列表加载与缓存，频道、播放列表和登录列表短时复用结果，减少重复请求。
- 优化 YouTube 播放历史恢复，尽量复用仍可用的直连播放地址，并避免未解析占位项污染历史记录。
- 优化 YouTube 11 位视频 id 识别，全局搜索、分类配置和历史恢复可直接处理裸视频 id。
- 优化 YouTube 页面和播放器交互，提升分类切换、内容打开和播放状态同步体验。

## 修复

- 修复 YouTube 频道标题在部分场景下显示不准确的问题。
- 修复 YouTube 清晰度切换时音轨选择丢失的问题。
- 修复腾讯元数据增强在部分搜索结果布局下匹配不到作品的问题。
