## 新增

- 新增 TVBox T4 JavaScript 爬虫运行支持，可加载和运行更多 TVBox JS 插件。
- 新增追更续播入口，追更条目可根据上次播放来源继续播放。
- 新增追更影片/剧集类型筛选与类型标识，追更列表可更清晰地区分电影、剧集和动漫。
- 新增更多豆瓣元数据字段展示，条目信息更完整。

## 优化

- 优化追更排序，优先按平台更新时间展示最新内容。
- 优化追更搜索结果和详情页布局，电影条目隐藏剧集列表，搜索结果展示更紧凑。
- 优化 TVBox JS 插件兼容性，补充 CommonJS、axios、Fastify 风格接口、UUID 和 JSON POST 等适配。
- 优化插件元数据匹配，减少平台来源误匹配。
- 优化元数据标题清理和权威类型保留，提升条目展示一致性。

## 修复

- 修复追更最新集数显示不准确的问题。
- 修复追更特别篇不可选择的问题。
- 修复多季追更未看更新数、最新季展示和后台刷新后 TMDB 季数据丢失的问题。
- 修复 TMDB 动漫类型被错误归类为剧集的问题。
- 修复插件操作区空布局导致的异常。
