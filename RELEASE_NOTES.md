## 新增

- 新增内置 YouTube 标签页：检测到系统 `yt-dlp` 可用时显示，支持内置分类、搜索、频道、播放列表、账号订阅/历史/稍后再看和播放恢复。
- 新增 YouTube 分类配置源，支持内置分类、远程 JSONC 和本地 JSONC；配置兼容 TVBox `class` / `filters` 结构，并支持频道、播放列表、视频 id 和 `LIST:` 关键词分类。
- 新增 YouTube 多音轨切换，支持在播放器中切换 `yt-dlp` 提供的不同语言音轨，并可配置默认音轨。
- 新增 YouTube 默认画质、默认字幕、元数据语言和地区设置。
- 新增弹幕可读性设置，支持调节透明度，减少画面遮挡并提升观看体验。
- 新增诊断信息导出，便于排查播放、环境和插件相关问题。
- 新增 YouTube 详情信息展示，补充 `VID`、频道、发布时间、时长、播放数、点赞数、评论数、分类、标签和简介。

## 优化

- 优化 YouTube 打开速度，点击卡片后先用已有标题和封面打开播放器，再在后台解析真实播放地址。
- 优化 YouTube 列表加载与缓存，频道、播放列表和登录列表短时复用结果，减少重复请求。
- 优化 YouTube 11 位视频 id 识别，全局搜索和历史恢复可直接处理裸视频 id。
- 优化 F1 系统信息与诊断展示，问题定位更直接。
- 优化新版爬虫签名兼容性，降低插件失效概率。

## 修复

- 修复部分 YouTube 字幕显示与加载问题，提升字幕播放稳定性。
- 修复 YouTube 清晰度切换时音轨选择丢失的问题。
