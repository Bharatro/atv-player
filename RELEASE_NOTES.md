## 新增

- 插件管理新增“批量导入”，支持直接输入 GitHub 仓库地址或 `spiders_v2.json` 地址，一次导入多份 `TvBox Python` 爬虫插件。
- 播放设置新增 YouTube 最高画质选项，可按需限制到 `480p`、`720p`、`1080p`、`1440p` 或 `2160p`。

## 优化

- 优化 YouTube 播放启动策略，默认以更稳妥的清晰度起播，通常能更快进入播放并减少首播阶段的反复缓冲。
- 优化 YouTube 清晰度切换体验，切换画质时更容易保留当前选择，重复解析同一地址时也会更快命中缓存。
- 优化多处界面交互：搜索框支持一键清空，插件管理打开后会自动聚焦搜索框，设置页与搜索输入框样式更统一。

## 修复

- 修复部分 YouTube 视频在字幕信息提取超时时容易卡住或起播失败的问题，超时后会自动重试播放地址提取。
- 修复部分 YouTube 清晰度切换失败后播放状态回退不完整的问题，避免切换失败后线路信息被错误清空。
